# 开发指南

## 仓库初始化

```sh
git clone https://github.com/remyzane/bpmn-js.git
cd bpmn-js
git remote add upstream  https://github.com/bpmn-io/bpmn-js.git
# 依赖库（../diagram-js-direct-editing/diagram-js-direct-editing.tgz）已添加到 package.json
# 找到 diagram-js-direct-editing-*.tgz 中版本号最高的文件（按自然版本排序）
npm add "file:$(ls ../diagram-js-direct-editing/diagram-js-direct-editing-*.tgz | sort -V | tail -n1)"
npm install
```

## 同步上游更新（bpmn-io/bpmn-js.git -> remyzane/bpmn-js.git）

```sh
git fetch upstream
git checkout main
git merge upstream/main
git push origin main
```

## 合并 main 分支（的上游更新）到 chinese 分支

```sh
git checkout chinese
git merge main
git push origin chinese
```

### 安装/更新依赖库

```bash
# 找到 diagram-js-direct-editing-*.tgz 中版本号最高的文件（按自然版本排序）
npm add "file:$(ls ../diagram-js-direct-editing/diagram-js-direct-editing-*.tgz | sort -V | tail -n1)"
```

## 打包发布

生成新的 bpmn-js.tgz，供其他项目直接使用，使用方法可参考 https://github.com/remyzane/vscode-zh-completion

```sh
git checkout chinese
npm add "file:$(ls ../diagram-js-direct-editing/diagram-js-direct-editing-*.tgz | sort -V | tail -n1)"
rm bpmn-js-*.tgz
npm pack
```
